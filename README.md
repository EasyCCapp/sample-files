# EasyCC Sample Files

Binary sample files downloaded by EasyCC when creating a new Sample Workspace.

These files demonstrate EasyCC's file type viewers (audio, video, fonts, 3D models, databases, ebooks, and archives). They are downloaded in the background after workspace creation — text-based samples are always available offline.
